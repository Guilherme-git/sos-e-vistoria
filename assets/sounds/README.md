# Sons do App

## Arquivo necessário

Adicione um arquivo de som chamado **`notification.mp3`** nesta pasta.

### Características recomendadas:
- **Formato:** MP3 ou WAV
- **Duração:** 1-2 segundos
- **Volume:** Moderado (será tocado a 50%)
- **Estilo:** Som de notificação suave (tipo "ding" ou "chime")

### Onde baixar sons gratuitos:
- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

### Exemplo de sons suaves:
- "Gentle notification"
- "Soft chime"
- "Pleasant ding"
- "Subtle bell"

**Após adicionar o arquivo `notification.mp3`, o som será tocado quando um novo chamado chegar!**
